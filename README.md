# Secret Santa Manager

A web application for organizing workplace gift exchanges with Slack integration.

## Features

- Participant management with address and wishlist
- Automatic Secret Santa matching with exclusion rules
- Slack bot for onboarding participants
- Customizable notification messages
- Timezone-aware gift reminders
- Password-protected admin access

## Tech Stack

- React + TypeScript frontend
- Express.js backend
- PostgreSQL database with Drizzle ORM
- Slack API integration
